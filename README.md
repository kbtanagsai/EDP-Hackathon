# Wireless Power Transfer Using Inductive Coupling

## Project Overview

This project demonstrates the basic principle of Wireless Power Transfer (WPT) using electromagnetic induction. Electrical energy is transmitted from a transmitter coil to a receiver coil without any direct electrical connection. The received energy is used to light an LED.

## Objective

To understand and demonstrate wireless transmission of electrical power using inductive coupling between two coils.

## Components Used

- Breadboard
- Copper Wire (for coils)
- LED
- Battery / Power Supply
- Switch
- Connecting Wires
- Resistor (if required)

## Working Principle

Wireless power transfer works on the principle of electromagnetic induction.

1. The transmitter coil is connected to a power source.
2. Current flowing through the transmitter coil generates a magnetic field.
3. The receiver coil is placed near the transmitter coil.
4. The changing magnetic field induces a voltage in the receiver coil.
5. The induced voltage powers the LED connected to the receiver coil.

## Circuit Setup

- Transmitter coil connected to the power source.
- Receiver coil connected to the LED.
- Both coils are aligned and placed close to each other.

## Results

The LED glows without any direct wired connection between the power source and the LED, demonstrating successful wireless power transmission.
